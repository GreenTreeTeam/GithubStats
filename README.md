## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/GreenTreeTeam/GithubStats.git
    ```
2. Install NPM packages

    ```sh
    yarn
    ```

    3.Start the app

```sh
  yarn start
```
